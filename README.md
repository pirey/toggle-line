# toggle-line.vim

Press `<Space>ts` to toggle statusline and tabline on / off simultaneously.

See `:h toggle-line`.

## Installation

Install using any vim plugin manager, or with vim-plug:

    Plug 'pirey/toggle-line.vim'

Then run `:PlugInstall`
